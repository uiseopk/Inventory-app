🔗 Firebase 연결
아래 5단계를 따라하면 여러 기기에서 실시간으로 데이터가 공유됩니다. 무료예요.
① Firebase 프로젝트 만들기
1. console.firebase.google.com 접속
2. 프로젝트 추가 클릭
3. 프로젝트 이름 입력 (예: 정육식당재고)
4. Google 애널리틱스 사용 안함 선택 후 완료
② 웹 앱 등록
1. 프로젝트 홈에서 </> (웹) 아이콘 클릭
2. 앱 닉네임 입력 (예: 재고앱)
3. Firebase SDK 설정값 복사해두기
③ Firestore 데이터베이스 생성
1. 왼쪽 메뉴 → Firestore Database
2. 데이터베이스 만들기 클릭
3. 테스트 모드로 시작 선택
4. 위치는 asia-northeast3 (Seoul) 선택
④ 아래에 설정값 입력
Firebase 콘솔 → 프로젝트 설정 → 내 앱 → firebaseConfig 에서 복사
설정값은 이 기기에만 저장됩니다
